# Emojis do Queridômetro

Esta pasta contém os ícones dos emojis usados no componente QueridometroModal.

## Arquivos necessários:

- `alvo.png` - 🎯
- `biscoito.png` - 🍪
- `cobra.png` - 🐍
- `coracao.png` - ❤️
- `coracao-partido.png` - 💔
- `mala.png` - 🧳
- `mentiroso.png` - 🤥
- `planta.png` - 🌱
- `vomito.png` - 🤮

## Especificações recomendadas:

- **Formato:** PNG com fundo transparente
- **Tamanho:** 32x32px ou 64x64px (será redimensionado)
- **Estilo:** Flat design, consistente com o app

## Como usar:

1. Adicione os arquivos PNG nesta pasta
2. No arquivo `QueridometroModal.tsx`, descomente os imports
3. Comente a versão Unicode e descomente a versão com imagens

Os emojis serão automaticamente redimensionados para 16x16px nos badges.
